WebRTC(supports Chrome browser)

WebRTC is an open source technology that supports web browsers for real-time voice and video calls. The advantage is that it does not require any plug-ins to be installed. It is ready to use and supports Mac OS and Windows. The disadvantage is that the currently compatible browsers are limited. Support Chrome58+, Firefox57+

Note: Other domestic browsers that use the chrome kernel have different support conditions due to the kernel version. They may be connected to WebRTC or may fail. At present, we have not done verification.

PC Agent (support Win7/10 system, Chrome, IE11/Edge browser)

PC Agent is a plug-in solution. The core lies in the need to install an audio and video plug-in locally. The advantage is that it supports more browsers, including Chrome55 and above, IE11 and the latest version of Edge. The disadvantage is that it only supports Windows systems and requires the installation of plug-ins.
Technical principle hint: The audio function in the whiteboard function requires the technical support of WebRTC or PCagent, but the whiteboard itself is not required and can be used separately.
